# استيراد / تصدير الطلاب — Excel

## Big O

| العملية | زمني | مكاني |
|---------|------|--------|
| استيراد n صف | O(n) | O(n) |
| تصدير | O(n) | O(n) |
| بحث حلقة بالاسم | O(1) مع خريطة محمّلة |

## عناوين Excel (بالترتيب)

1. الاسم الرباعي  
2. الهوية  
3. الجنسية  
4. رقم الجوال  
5. المدرسة  
6. الصف  
7. مقدار الحفظ  
8. رقم جوال ولي الأمر  
9. هوية ولي الأمر إن وجدت  
10. الحلقة  
11. أعراض صحية إذا وجد  

## أوضاع الاستيراد

| الوضع | الوظيفة |
|--------|---------|
| **تسجيل جماعي** | إضافة طالب جديد أو تحديث موجود (بالهوية/الجوال) + حلقة |
| **نقل جماعي** | نقل تراكمي بالهوية/الجوال إلى حلقة جديدة |
